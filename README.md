<h1 align="center">Personal Introduction Webpage</h1>

<h2 align="center">Project Overview</h2>

This project is a personal single-page website created using semantic HTML and an external CSS stylesheet. The purpose of this assignment is to practice structuring a webpage, applying styling with CSS, and using a professional color palette to improve the visual appearance of the site.

The webpage introduces who I am, my career goals, hobbies, and interests, along with a link to my GitHub profile.

---

<h2 align="center">Features</h2>

This webpage includes:

- A main header with my name and professional interests
- A Career Goals section with two descriptive paragraphs
- A Hobbies and Interests section using a bulleted list
- A GitHub profile link
- A footer with project information
- A custom color palette applied using CSS
- Centered layout and styled sections for improved readability

---

<h2 align="center">Files Included</h2>

This project contains the following files:

- `index.html` — The main webpage created using semantic HTML  
- `style.css` — The external stylesheet used to apply colors, spacing, and layout styling  
- `README.md` — This file describing the project  

---

<h2 align="center">Technologies Used</h2>

- HTML5 (Semantic Elements)  
- CSS3 (External Stylesheet)  
- Google Arts & Culture Color Palette Tool  

Color palette used:

- `#E6C9A7`
